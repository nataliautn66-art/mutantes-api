   Detección de Mutantes API (Trabajo Práctico Global - UTN)

Este proyecto implementa una API RESTful para la detección de mutantes basado en el análisis de secuencias de ADN, cumpliendo con los requisitos de la consigna de Nivel 3 del Examen Global.

  Tecnologías Utilizadas

* **Lenguaje:** Java (versión 17 o superior)
* **Framework:** Spring Boot 3+
* **Base de Datos:** H2 Database (Embebida en memoria)
* **Persistencia:** Spring Data JPA
* **Construcción:** Gradle

    Pre-requisitos

Para ejecutar la API localmente, necesitas tener instalado:

1.  **Java Development Kit (JDK):** Versión 17 o superior.
2.  **Gradle:** Herramienta de automatización de construcción (se recomienda usar el wrapper `./gradlew` incluido).
3.  **Git:** Para clonar el repositorio.
